# B. Массив эргүүлэх

## Бодлого

**N** элементтэй массив болон **Q** асуулт өгөгдөнө. Асуулт бүрд **L** ба **R** өгөгдөх бөгөөд массивын **L**-ээс **R** хүртэлх хэсгийг эргүүлэх (reverse) ёстой. Бүх асуултыг гүйцэтгэсний дараа массивыг хэвлэ.

## Оролт

Эхний мөрөнд **N** ба **Q** — массивын хэмжээ ба асуултын тоо.

Хоёр дахь мөрөнд **N** бүхэл тоо — массивын элементүүд.

Дараагийн **Q** мөр бүрд **L** ба **R** (1-indexed) — эргүүлэх хэсгийн эхлэл ба төгсгөл.

## Гаралт

Бүх асуултыг гүйцэтгэсний дараах массивыг зайгаар тусгаарлан хэвлэ.

## Хязгаарлалт

- Цагийн хязгаар: **2 секунд**
- Санах ойн хязгаар: **256 MB**
- 1 ≤ L ≤ R ≤ N

## Subtask

| Subtask | Оноо | Нөхцөл |
|---------|-------|--------|
| 1 | 30 | N ≤ 10, Q = 1 |
| 2 | 30 | N ≤ 1000, Q ≤ 1000 |
| 3 | 40 | N ≤ 100000, Q ≤ 100000 |

## Жишээ

### Жишээ 1
**Оролт:**
```
5 2
1 2 3 4 5
2 4
1 3
```
**Гаралт:**
```
3 2 1 4 5
```

### Жишээ 2
**Оролт:**
```
3 1
7 8 9
1 3
```
**Гаралт:**
```
9 8 7
```

### Тайлбар

Жишээ 1:
- Эхний массив: [1, 2, 3, 4, 5]
- Асуулт 1 (L=2, R=4): 2-4 хэсгийг эргүүлнэ → [1, 4, 3, 2, 5]
- Асуулт 2 (L=1, R=3): 1-3 хэсгийг эргүүлнэ → [3, 2, 1, 4, 5]

Жишээ 2: Бүх массивыг эргүүлнэ → [9, 8, 7]
