# C. Тэмдэгтийн давтамж

## Бодлого

Тэмдэгт мөр **S** өгөгдөнө. **S** дотор хамгийн олон удаа давтагдсан тэмдэгтийг ол. Хэрэв хэд хэдэн тэмдэгт ижил хамгийн их давтамжтай бол тэдгээрийг **цагаан толгойн дарааллаар** хэвлэ.

Мөн хамгийн олон давтагдсан тэмдэгтийн давтамжийг хэвлэ.

## Оролт

Эхний мөрөнд бүхэл тоо **T** — асуултын тоо.

Дараагийн **T** мөр бүрд тэмдэгт мөр **S** өгөгдөнө. S нь зөвхөн англи жижиг үсгээс бүрдэнэ.

## Гаралт

Асуулт бүрд хоёр мөр хэвлэ:
1. Хамгийн олон давтагдсан тэмдэгтүүдийг цагаан толгойн дарааллаар зайгүй хэвлэ
2. Тэдгээрийн давтамжийг хэвлэ

## Хязгаарлалт

- Цагийн хязгаар: **2 секунд**
- Санах ойн хязгаар: **256 MB**

## Subtask

| Subtask | Оноо | Нөхцөл |
|---------|-------|--------|
| 1 | 20 | T = 1, |S| ≤ 20, нэг л хамгийн их давтамжтай тэмдэгт байна |
| 2 | 30 | T = 1, |S| ≤ 1000 |
| 3 | 50 | 1 ≤ T ≤ 1000, |S| ≤ 100000, нийт тэмдэгтийн нийлбэр ≤ 500000 |

## Жишээ

### Жишээ 1
**Оролт:**
```
3
aabbc
hello
zzz
```
**Гаралт:**
```
ab
2
l
2
z
3
```

### Тайлбар

Асуулт 1: "aabbc" — 'a' 2 удаа, 'b' 2 удаа, 'c' 1 удаа. Хамгийн их = 2. 'a' ба 'b' хоёулаа 2 удаа → "ab".

Асуулт 2: "hello" — 'h' 1, 'e' 1, 'l' 2, 'o' 1. Хамгийн их = 2. Зөвхөн 'l' → "l".

Асуулт 3: "zzz" — 'z' 3 удаа → "z", давтамж 3.
