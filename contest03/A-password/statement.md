# A. Нууц үг шалгах

## Бодлого

Болд нэвтрэх системд нууц үг шалгах функц бичих даалгавар авав. Нууц үг нь **хүчтэй** гэж тооцогдохын тулд дараах **бүх** нөхцлийг хангасан байх ёстой:

1. Урт нь хамгийн багадаа **8** тэмдэгт байх
2. Дор хаяж **1** том үсэг ('A'-'Z') агуулсан байх
3. Дор хаяж **1** жижиг үсэг ('a'-'z') агуулсан байх
4. Дор хаяж **1** цифр ('0'-'9') агуулсан байх

Хэрэв бүх нөхцлийг хангасан бол **"STRONG"**, үгүй бол **"WEAK"** гэж хэвлэ.

## Оролт

Нэг мөрөнд тэмдэгт мөр **S** өгөгдөнө. S нь зөвхөн англи том, жижиг үсэг болон цифрээс бүрдэнэ.

## Гаралт

**"STRONG"** эсвэл **"WEAK"** гэж хэвлэ.

## Хязгаарлалт

- Цагийн хязгаар: **1 секунд**
- Санах ойн хязгаар: **256 MB**
- 1 ≤ |S| ≤ 100

## Subtask

| Subtask | Оноо | Нөхцөл |
|---------|-------|--------|
| 1 | 30 | |S| ≤ 7 (урт нь 8-аас бага → үргэлж WEAK) |
| 2 | 30 | S нь зөвхөн жижиг үсэг ба цифрээс бүрдэнэ |
| 3 | 40 | Бүх тохиолдол |

## Жишээ

### Жишээ 1
**Оролт:**
```
Hello123
```
**Гаралт:**
```
STRONG
```

### Жишээ 2
**Оролт:**
```
hello
```
**Гаралт:**
```
WEAK
```

### Жишээ 3
**Оролт:**
```
abcdefgh
```
**Гаралт:**
```
WEAK
```

### Тайлбар

Жишээ 1: Урт 8, том үсэг ('H'), жижиг үсэг ('e','l','o'), цифр ('1','2','3') → STRONG.

Жишээ 2: Урт 5 < 8 → WEAK.

Жишээ 3: Урт 8, жижиг үсэг байна, гэхдээ том үсэг ба цифр алга → WEAK.
