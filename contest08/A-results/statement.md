# A. Тэмцээний үр дүн

## Бодлого

Програмчлалын тэмцээнд **N** оролцогч оролцов. Оролцогч бүрийн нэр ба оноо өгөгдөнө. Оролцогчдыг оноогоор **буурах** дарааллаар эрэмбэлж, байр бүрийг хэвлэ. Хэрэв хоёр оролцогчийн оноо тэнцүү бол нэрний **цагаан толгойн** дарааллаар эрэмбэл.

## Оролт

Эхний мөрөнд **N** — оролцогчдын тоо.

Дараагийн **N** мөр бүрд оролцогчийн нэр ба оноо зайгаар тусгаарлагдсан.

## Гаралт

**N** мөрөнд байр, нэр, оноог зайгаар тусгаарлан хэвлэ.

## Хязгаарлалт

- Цагийн хязгаар: **1 секунд**
- Санах ойн хязгаар: **256 MB**

## Subtask

| Subtask | Оноо | Нөхцөл |
|---------|-------|--------|
| 1 | 30 | N ≤ 100 |
| 2 | 30 | N ≤ 10000 |
| 3 | 40 | N ≤ 200000 |

## Жишээ

### Жишээ 1
**Оролт:**
```
5
Alice 90
Bob 85
Charlie 90
Diana 95
Eve 85
```
**Гаралт:**
```
1 Diana 95
2 Alice 90
3 Charlie 90
4 Bob 85
5 Eve 85
```

### Тайлбар

- Diana хамгийн их оноотой (95) → 1-р байр
- Alice ба Charlie хоёулаа 90 оноотой. Цагаан толгойн дарааллаар Alice < Charlie → Alice 2-р, Charlie 3-р
- Bob ба Eve хоёулаа 85 оноотой. Bob < Eve → Bob 4-р, Eve 5-р
