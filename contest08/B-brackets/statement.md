# B. Хаалт шалгах

## Бодлого

Хаалтнуудаас бүтсэн тэмдэгт мөр өгөгдөнө. Мөр нь зөв хаагдсан эсэхийг шалга.

Зөв хаагдсан мөр гэдэг нь:
- Хоосон мөр нь зөв
- Хэрэв S зөв бол `(S)`, `[S]`, `{S}` нь зөв
- Хэрэв A, B хоёулаа зөв бол AB нь зөв

## Оролт

Эхний мөрөнд **T** — тестийн тоо.

Дараагийн **T** мөр бүрд хаалтнуудаас бүтсэн тэмдэгт мөр өгөгдөнө.

## Гаралт

Тест бүрд **"YES"** (зөв хаагдсан) эсвэл **"NO"** (зөв хаагдаагүй) гэж хэвлэ.

## Хязгаарлалт

- Цагийн хязгаар: **1 секунд**
- Санах ойн хязгаар: **256 MB**

## Subtask

| Subtask | Оноо | Нөхцөл |
|---------|-------|--------|
| 1 | 30 | T ≤ 10, мөрийн урт ≤ 100, зөвхөн `()` |
| 2 | 30 | T ≤ 100, мөрийн урт ≤ 1000, `()`, `[]`, `{}` |
| 3 | 40 | T ≤ 1000, мөрийн урт ≤ 100000 |

## Жишээ

### Жишээ 1
**Оролт:**
```
5
()
([{}])
(]
((())
({[}])
```
**Гаралт:**
```
YES
YES
NO
NO
NO
```

### Тайлбар

- `()` — зөв
- `([{}])` — зөв (гүнзгий хаалт)
- `(]` — ( нээгээд ] хаасан → буруу төрөл
- `((())` — нээсэн хаалт илүүдсэн
- `({[}])` — `[` нээгээд `}` хаасан → буруу
