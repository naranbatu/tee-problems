# A. Секундыг хөрвүүлэх

## Бодлого

**S** секунд өгөгдсөн бол хэдэн **цаг**, хэдэн **минут**, хэдэн **секунд** болохыг тодорхойл.

## Оролт

Нэг бүхэл тоо **S** — нийт секундын тоо.

## Гаралт

Нэг мөрөнд гурван тоог зайгаар тусгаарлан хэвлэ: **цаг**, **минут**, **секунд**.

## Хязгаарлалт

- Цагийн хязгаар: **1 секунд**
- Санах ойн хязгаар: **256 MB**

## Subtask

| Subtask | Оноо | Нөхцөл |
|---------|-------|--------|
| 1 | 30 | 0 ≤ S ≤ 59 |
| 2 | 30 | 0 ≤ S ≤ 100000 |
| 3 | 40 | 0 ≤ S ≤ 10^18 |

## Жишээ

### Жишээ 1
**Оролт:**
```
3661
```
**Гаралт:**
```
1 1 1
```

### Жишээ 2
**Оролт:**
```
45
```
**Гаралт:**
```
0 0 45
```

### Тайлбар

Жишээ 1: 3661 секунд = 1 цаг, 1 минут, 1 секунд.

Жишээ 2: 45 секунд = 0 цаг, 0 минут, 45 секунд.
