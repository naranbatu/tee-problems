# C. Анхны тоо

## Бодлого

**T** асуулт өгөгдөнө. Асуулт бүрд **N** тоо өгөгдөх бөгөөд уг тоо **анхны тоо** мөн эсэхийг тодорхойл.

Анхны тоо гэдэг нь 1-ээс их, зөвхөн 1 болон өөртөө хуваагддаг тоо юм. Жишээ нь: 2, 3, 5, 7, 11, 13 нь анхны тоонууд.

## Оролт

Эхний мөрөнд **T** — асуултын тоо.

Дараагийн **T** мөр бүрд нэг бүхэл тоо **N** өгөгдөнө.

## Гаралт

Асуулт бүрийн хариуд **"YES"** (анхны тоо) эсвэл **"NO"** (анхны тоо биш) гэж хэвлэ.

## Хязгаарлалт

- Цагийн хязгаар: **2 секунд**
- Санах ойн хязгаар: **256 MB**

## Subtask

| Subtask | Оноо | Нөхцөл |
|---------|-------|--------|
| 1 | 20 | T = 1, 1 ≤ N ≤ 100 |
| 2 | 30 | T = 1, 1 ≤ N ≤ 10^6 |
| 3 | 50 | 1 ≤ T ≤ 10000, 1 ≤ N ≤ 10^9 |

## Жишээ

### Жишээ 1
**Оролт:**
```
3
7
4
1
```
**Гаралт:**
```
YES
NO
NO
```

### Тайлбар

- 7 нь анхны тоо (зөвхөн 1 ба 7-д хуваагдана).
- 4 нь анхны тоо биш (4 = 2 × 2).
- 1 нь анхны тоо биш (тодорхойлолтоор 1-ээс их байх ёстой).
