# B. Агуулахын бүртгэл

## Бодлого

Агуулахад бараа нэмэх, хасах, тоо ширхэг шалгах, хамгийн их бараатай төрлийг олох гэсэн үйлдлүүдийг гүйцэтгэ.

- **ADD name count** — name нэртэй бараанаас count ширхэг нэмнэ
- **REMOVE name count** — name нэртэй бараанаас count ширхэг хасна. Хэрэв хүрэлцэхгүй бол **"NOT_ENOUGH"** хэвлэ. Хэрэв 0 болвол бүртгэлээс арилгана.
- **COUNT name** — name нэртэй бараа хэдэн ширхэг байгааг хэвлэ (байхгүй бол 0)
- **MAX** — Хамгийн олон ширхэгтэй барааны нэр ба тоог хэвлэ. Хоосон бол **"EMPTY"** хэвлэ. Тэнцүү бол цагаан толгойн дарааллаар эхнийхийг хэвлэ.

## Оролт

Эхний мөрөнд **Q** — үйлдлийн тоо.

Дараагийн **Q** мөр бүрд нэг үйлдэл.

## Гаралт

COUNT, MAX, мөн NOT_ENOUGH тохиолдлуудад хариуг хэвлэ.

## Хязгаарлалт

- Цагийн хязгаар: **2 секунд**
- Санах ойн хязгаар: **256 MB**

## Subtask

| Subtask | Оноо | Нөхцөл |
|---------|-------|--------|
| 1 | 30 | Q ≤ 100, зөвхөн ADD ба COUNT |
| 2 | 30 | Q ≤ 10000 |
| 3 | 40 | Q ≤ 500000, count ≤ 10^9 |

## Жишээ

### Жишээ 1
**Оролт:**
```
8
ADD apple 10
ADD banana 5
COUNT apple
MAX
REMOVE apple 7
COUNT apple
REMOVE apple 5
MAX
```
**Гаралт:**
```
10
apple 10
3
NOT_ENOUGH
banana 5
```

### Тайлбар

- ADD apple 10, ADD banana 5
- COUNT apple → 10
- MAX → apple 10
- REMOVE apple 7 → apple = 3
- COUNT apple → 3
- REMOVE apple 5 → 3 < 5 → NOT_ENOUGH
- MAX → apple 3 vs banana 5 → banana 5
