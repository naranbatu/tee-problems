# C. Медиан олох

## Бодлого

**N** тоо өгөгдөнө. **Q** асуулт бүрд **L** ба **R** (1-indexed) өгөгдөх бөгөөд массивын L-ээс R хүртэлх хэсгийн **медианыг** ол.

Медиан гэдэг нь тоонуудыг өсөх дарааллаар эрэмбэлсний дараа дунд байрлалд буй тоо. Хэрэв тоонуудын тоо тэгш бол хоёр дунд тооны **бага** нь медиан юм.

## Оролт

Эхний мөрөнд **N** ба **Q**.

Хоёр дахь мөрөнд **N** бүхэл тоо.

Дараагийн **Q** мөр бүрд **L** ба **R**.

## Гаралт

Асуулт бүрд медианыг хэвлэ.

## Хязгаарлалт

- Цагийн хязгаар: **3 секунд**
- Санах ойн хязгаар: **256 MB**

## Subtask

| Subtask | Оноо | Нөхцөл |
|---------|-------|--------|
| 1 | 20 | N, Q ≤ 100 |
| 2 | 30 | N ≤ 1000, Q ≤ 1000 |
| 3 | 50 | N ≤ 100000, Q ≤ 100000 |

## Жишээ

### Жишээ 1
**Оролт:**
```
5 3
3 1 4 1 5
1 5
2 4
1 2
```
**Гаралт:**
```
3
1
1
```

### Тайлбар

- [3,1,4,1,5] эрэмбэлвэл [1,1,3,4,5] → медиан = 3 (3-р байрлал, 5 элемент)
- [1,4,1] эрэмбэлвэл [1,1,4] → медиан = 1
- [3,1] эрэмбэлвэл [1,3] → хоёр дунд тооны бага нь = 1
