# A. Давтагдаагүй тоонууд

## Бодлого

**N** тоо өгөгдөнө. Эдгээр тоонуудаас **давтагдаагүй** (нэг л удаа тааралдсан) тоонуудыг **өсөх дарааллаар** хэвлэ.

## Оролт

Эхний мөрөнд **N** — тоонуудын тоо.

Хоёр дахь мөрөнд **N** бүхэл тоо өгөгдөнө.

## Гаралт

Давтагдаагүй тоонуудыг өсөх дарааллаар зайгаар тусгаарлан хэвлэ. Хэрэв нэг ч давтагдаагүй тоо байхгүй бол **"EMPTY"** гэж хэвлэ.

## Хязгаарлалт

- Цагийн хязгаар: **1 секунд**
- Санах ойн хязгаар: **256 MB**

## Subtask

| Subtask | Оноо | Нөхцөл |
|---------|-------|--------|
| 1 | 30 | N ≤ 10, тоонууд 1..100 |
| 2 | 30 | N ≤ 1000, тоонууд -10^6..10^6 |
| 3 | 40 | N ≤ 200000, тоонууд -10^9..10^9 |

## Жишээ

### Жишээ 1
**Оролт:**
```
6
3 1 4 1 5 3
```
**Гаралт:**
```
4 5
```

### Жишээ 2
**Оролт:**
```
3
7 7 7
```
**Гаралт:**
```
EMPTY
```

### Тайлбар

Жишээ 1: 1 болон 3 давтагдсан. 4 ба 5 нэг л удаа тааралдсан → өсөх дарааллаар: 4 5.

Жишээ 2: Бүх тоо давтагдсан → EMPTY.
