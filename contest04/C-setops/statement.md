# C. Олонлогийн үйлдлүүд

## Бодлого

Хоосон олонлог **S** өгөгдсөн ба **Q** асуулт хариулна. Асуулт бүр дараах 4 төрлийн аль нэг байна:

- **ADD x** — S олонлогт x тоог нэмнэ
- **REMOVE x** — S олонлогоос x тоог хасна (байхгүй бол юу ч хийхгүй)
- **CHECK x** — x тоо S олонлогт байгаа эсэхийг шалгаж **"YES"** эсвэл **"NO"** гэж хэвлэ
- **COUNT** — S олонлогт хэдэн элемент байгааг хэвлэ

## Оролт

Эхний мөрөнд **Q** — асуултын тоо.

Дараагийн **Q** мөр бүрд нэг асуулт өгөгдөнө.

## Гаралт

CHECK ба COUNT асуулт бүрд хариуг тус тусад нь мөр мөрөөр хэвлэ.

## Хязгаарлалт

- Цагийн хязгаар: **2 секунд**
- Санах ойн хязгаар: **256 MB**

## Subtask

| Subtask | Оноо | Нөхцөл |
|---------|-------|--------|
| 1 | 20 | Q ≤ 100, зөвхөн ADD ба CHECK |
| 2 | 30 | Q ≤ 10000, бүх төрлийн асуулт |
| 3 | 50 | Q ≤ 500000, -10^9 ≤ x ≤ 10^9 |

## Жишээ

### Жишээ 1
**Оролт:**
```
8
ADD 5
ADD 3
ADD 5
CHECK 5
CHECK 7
COUNT
REMOVE 5
COUNT
```
**Гаралт:**
```
YES
NO
2
1
```

### Тайлбар

- ADD 5 → S = {5}
- ADD 3 → S = {3, 5}
- ADD 5 → S = {3, 5} (аль хэдийн байгаа)
- CHECK 5 → YES
- CHECK 7 → NO
- COUNT → 2
- REMOVE 5 → S = {3}
- COUNT → 1
