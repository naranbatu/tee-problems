# A. Хамгийн ойр хос

## Бодлого

**N** бүхэл тоо өгөгдөнө. Хоёр **ялгаатай байрлал** дахь тоонуудын ялгааны абсолют утгын хамгийн бага утгыг ол.

Өөрөөр хэлбэл, `|a[i] - a[j]|` (i ≠ j) утгуудын хамгийн багыг ол.

## Оролт

Эхний мөрөнд **N**.

Хоёр дахь мөрөнд **N** бүхэл тоо.

## Гаралт

Нэг тоо — хамгийн бага ялгаа.

## Хязгаарлалт

- Цагийн хязгаар: **1 секунд**
- Санах ойн хязгаар: **256 MB**

## Subtask

| Subtask | Оноо | Нөхцөл |
|---------|-------|--------|
| 1 | 30 | N ≤ 100 |
| 2 | 30 | N ≤ 10000 |
| 3 | 40 | N ≤ 500000, тоонууд ≤ 10^9 |

## Жишээ

### Жишээ 1
**Оролт:**
```
5
4 9 1 7 2
```
**Гаралт:**
```
1
```

### Тайлбар

Эрэмбэлвэл: [1, 2, 4, 7, 9]. Зэргэлдээ тоонуудын ялгаа: |2-1|=1, |4-2|=2, |7-4|=3, |9-7|=2. Хамгийн бага нь **1**.

### Жишээ 2
**Оролт:**
```
3
5 5 5
```
**Гаралт:**
```
0
```
