# B. Үхрүүдийг байрлуулах

## Бодлого

**N** хашаа шугаман дээр байрлаж байна. Хашаа бүр тодорхой координаттай (бүхэл тоо). **C** үхрийг хашаанд байрлуулах ёстой бөгөөд ямар нэг хоёр үхрийн хоорондох хамгийн бага зай аль болох **их** байхаар байрлуул.

Энэ хамгийн бага зайн хамгийн их утгыг ол.

## Оролт

Эхний мөрөнд **N** ба **C** — хашааны тоо ба үхрийн тоо.

Хоёр дахь мөрөнд **N** бүхэл тоо — хашааны координатууд.

## Гаралт

Нэг тоо — хамгийн бага зайн хамгийн их боломжит утга.

## Хязгаарлалт

- Цагийн хязгаар: **2 секунд**
- Санах ойн хязгаар: **256 MB**
- 2 ≤ C ≤ N

## Subtask

| Subtask | Оноо | Нөхцөл |
|---------|-------|--------|
| 1 | 30 | N ≤ 10, координатууд ≤ 100 |
| 2 | 30 | N ≤ 1000, координатууд ≤ 10^6 |
| 3 | 40 | N ≤ 200000, координатууд ≤ 10^9 |

## Жишээ

### Жишээ 1
**Оролт:**
```
5 3
1 2 8 4 9
```
**Гаралт:**
```
3
```

### Жишээ 2
**Оролт:**
```
4 2
10 30 20 40
```
**Гаралт:**
```
30
```

### Тайлбар

Жишээ 1: Координатуудыг эрэмбэлвэл: 1, 2, 4, 8, 9. 3 үхрийг 1, 4, 8-д байрлуулбал хамгийн бага зай = min(4-1, 8-4) = 3. Энэ нь хамгийн их боломжит утга.

Жишээ 2: Эрэмбэлсэн: 10, 20, 30, 40. 2 үхрийг 10 ба 40-д байрлуулвал зай = 30.
