# C. Хамгийн бага хамгийн их

## Бодлого

**N** тооноос бүрдсэн массив ба **K** тоо өгөгдөнө. Массивыг **K** дараалсан хэсэгт хуваана (хэсэг бүр хамгийн багадаа 1 элементтэй). Хэсэг бүрийн **нийлбэрийн** хамгийн ихийг аль болох **бага** болгох ёстой.

Хэсгүүдийн нийлбэрийн хамгийн ихийн хамгийн бага боломжит утгыг ол.

## Оролт

Эхний мөрөнд **N** ба **K**.

Хоёр дахь мөрөнд **N** эерэг бүхэл тоо — массивын элементүүд.

## Гаралт

Нэг тоо — хариу.

## Хязгаарлалт

- Цагийн хязгаар: **2 секунд**
- Санах ойн хязгаар: **256 MB**
- 1 ≤ K ≤ N

## Subtask

| Subtask | Оноо | Нөхцөл |
|---------|-------|--------|
| 1 | 20 | N ≤ 10, K ≤ N |
| 2 | 30 | N ≤ 1000, элемент бүр ≤ 10^6 |
| 3 | 50 | N ≤ 200000, элемент бүр ≤ 10^9 |

## Жишээ

### Жишээ 1
**Оролт:**
```
5 3
2 3 5 1 4
```
**Гаралт:**
```
5
```

### Жишээ 2
**Оролт:**
```
4 1
1 2 3 4
```
**Гаралт:**
```
10
```

### Тайлбар

Жишээ 1: [2, 3] [5] [1, 4] → нийлбэрүүд: 5, 5, 5 → хамгийн их = 5. Энэ нь хамгийн бага боломжит.

Жишээ 2: K=1 бол бүх элемент нэг хэсэгт → нийлбэр = 10.
