# C. Орчуулагч

## Бодлого

Хоёр хэлний толь бичиг өгөгдсөн: **N** үгийн хос (үг_А, үг_Б). Дараа нь **Q** асуулт өгөгдөх бөгөөд, асуулт бүрд нэг үг өгөгдвөл нөгөө хэл дээрх орчуулгыг хэвлэ.

Орчуулга хоёр чиглэлд ажиллана: үг_А → үг_Б, мөн үг_Б → үг_А.

Хэрэв толь бичигт байхгүй үг асуувал **"UNKNOWN"** гэж хэвлэ.

## Оролт

Эхний мөрөнд **N** ба **Q** — толь бичгийн хэмжээ ба асуултын тоо.

Дараагийн **N** мөр бүрд хоёр үг: **үг_А** ба **үг_Б** (зайгаар тусгаарлан).

Дараагийн **Q** мөр бүрд нэг үг — орчуулах үг.

## Гаралт

Асуулт бүрийн хариуг мөр мөрөөр хэвлэ.

## Хязгаарлалт

- Цагийн хязгаар: **2 секунд**
- Санах ойн хязгаар: **256 MB**
- Үг бүр зөвхөн англи жижиг үсгээс бүрдэнэ, урт ≤ 20

## Subtask

| Subtask | Оноо | Нөхцөл |
|---------|-------|--------|
| 1 | 20 | N, Q ≤ 10, зөвхөн үг_А → үг_Б чиглэл |
| 2 | 30 | N, Q ≤ 1000 |
| 3 | 50 | N ≤ 200000, Q ≤ 500000 |

## Жишээ

### Жишээ 1
**Оролт:**
```
3 5
hello sain
cat muur
dog nokhoi
hello
muur
bird
nokhoi
sain
```
**Гаралт:**
```
sain
cat
UNKNOWN
dog
hello
```

### Тайлбар

- "hello" → толинд байна → "sain"
- "muur" → толинд байна (урвуу) → "cat"
- "bird" → толинд алга → "UNKNOWN"
- "nokhoi" → урвуу → "dog"
- "sain" → урвуу → "hello"
